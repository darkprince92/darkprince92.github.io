As a Graduate Research Assistant at UTA, I work in the SAFE Lab under Dr. Allison Sullivan. My research focuses on making formal specification languages — particularly Alloy, a model-finding language based on relational logic — more accessible and usable for software engineers and novice users.

The central theme of my work is bringing live programming and AI-assisted tooling to formal methods. Formal specification is powerful but notoriously hard to adopt; users struggle with cryptic error messages, long feedback cycles, and a steep learning curve. My goal is to collapse that feedback loop and lower the barrier to entry.

**Live feedback for Alloy (FormaliSE 2025):** I designed and built a system that provides real-time diagnostic feedback as users write Alloy models. Rather than waiting for a full model-finding run, the system surfaces counterexamples and structural feedback incrementally, reducing the iterative debugging cycle. This work was published at FormaliSE 2025 (IEEE). I collaborated with Aleksandar Jovanovic and Dr. Sullivan on this.

**Formula completion for Alloy:** I developed a formula completion system for Alloy that provides intelligent suggestions while users write logical constraints. The system uses syntactic and semantic analysis to understand partially written formulas and synthesize valid completions — analogous to code completion for a formal specification language. This work is under review.

**VS Code language server extension:** I built a full Language Server Protocol (LSP) extension for Alloy in VS Code using Lsp4j and Java. The extension provides context-aware code completion, syntax error highlighting, and error reporting directly in the editor. This is available as an open-source tool for Alloy users.

**LLM-assisted specification generation:** My current direction investigates using large language models to help users write Alloy specifications from informal natural-language requirements. I am exploring prompting strategies and fine-tuning approaches to bridge natural language and formal logic — combining NLP with program synthesis for formal methods.

This research combines my industry background in software tooling with academic rigor, producing artifacts that are both publishable and practically useful.
