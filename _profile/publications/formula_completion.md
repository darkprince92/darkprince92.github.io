This paper presents a formula completion system for Alloy, the relational software modeling language. The goal is to help users complete partially written logical formulas by suggesting syntactically valid and semantically meaningful continuations — analogous to code completion in general-purpose IDEs, but applied to formal specification.

The approach uses syntactic analysis to understand the structure of a partially written formula, and semantic analysis to reason about the types and relations in scope, producing ranked completion suggestions that guide the user toward a valid and useful formula.

This is primary-authored by me (first author), in collaboration with Imtiaz Uddin and Dr. Allison Sullivan. The paper is currently under review.
