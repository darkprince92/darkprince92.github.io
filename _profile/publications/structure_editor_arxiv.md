This preprint presents a structure editor for building software models in Alloy. Structure editors (also called projectional editors) constrain the editing surface to syntactically valid constructs, preventing syntax errors entirely rather than detecting them after the fact. This is particularly relevant for formal specification languages, where the grammar is precise and users frequently struggle with syntax.

The paper describes the design and implementation of a structure editor tailored for Alloy models, exploring how this editing paradigm can lower the learning curve and reduce friction for users writing formal specifications.

I led this work as first author, with contributions from Aleksandar Jovanovic and Dr. Allison Sullivan.
