\begin{rSection}{Projects and Services}

\begin{rBullets}
    \item {\bf Nationwide Quiz:} Designed and optimized a nationwide quiz system on AWS, preloading 50M questions via a data pipeline using S3, Lambda, and DynamoDB. Served 100K concurrent users through a load-balanced architecture, using Redis to fetch keys and DynamoDB for quick data access.

    \item {\bf User verification automation:} Validated the photo and National ID of the user against their data in the national ID database and verified users automatically in the iPay user verification procedure. Technologies: \href{https://github.com/tesseract-ocr/tesseract}{Tesseract}, \href{https://github.com/ageitgey/face_recognition}{Face Recognition}, \href{https://github.com/apache/airflow}{Airflow}.
    
    \item {\bf Data warehouse:} Built a streaming data pipeline for shipping data and logs using \href{https://github.com/confluentinc/kafka-connect-jdbc}{Kafka Connect} from different data sources to a data warehouse with retention and backups and deployed as a distributed platform to access and query data on Elasticsearch and PostgreSQL.
    
    \item {\bf Instant cashback:} Engineered end to end pipeline for instant cashback service built with groovy DSL based rules running on top of event streams. This is built by streaming events through Kafka and processing events with distributed rule engines.

\end{rBullets}
